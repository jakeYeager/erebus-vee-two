# Project Naming And Organization Rules

- The project is divided into many research "topics" that will their own directory to isolate context
- Each topic directory will be located in the project root
- Other resource directories will be located in the project root to share resources between projects, i.e. `data/`, `lib/` etc.
- Each topic will have its' own subcontext `CLAUDE.md`
- Each topic will have an identifier 
- The topic identifier will be used in file, directory, and documention block names, i.e. `topic-<identifier>`, `# Topic [identifier]`
- Each topic will contain cases relivant to their topics' subject
- Each case will be given an identifier 
- The case identifier will be used in file, directory, and documention blocks name, i.e. `case-<identifier>`, `# Case [identifier]`
